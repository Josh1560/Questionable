Rules

By using Questionable, you agree abide by these rules.

Disobeying these rules will result in your server being blacklisted.

Do not use the bot for racism, homophobia or any other offensive purpose.
Do not use macros, or spam the bot
Never share any bugs, please tell a developer in our server.
Don't try to host Questionable yourself.

Commands

Below you can find a full list of commands.

You cannot use commands in direct messages.
  • What is a command?
    • It's a specific function of the bot that can be used with q/.
  • What is an alias?
    • It's another way to use the command.
  • What are the asterisks for?
    • An asterisk shows that an argument is required.
  • Why can't I use developer commands?
    • Simply because they offer too much power for an average user.
  • Should I create a clone of Questionable?
    • I don't suggest cloning the GitHub repo at all, as this will get you blacklisted.
      You'd also be wasting your time and money.
      But if you wanna create a clone of Questionable from scratch, go ahead.
  • What library does Questionable use?
    • Believe it or not, it uses discord.py.
  • How do you host your bot?
    • Secret methods.

Basic

These are basic commands that you should expect with most bots.

Command Alias Description
ping None Calculate the bot's ping in milliseconds.
help None Displays a link to this website.
invite None Get the bot's invite link.
vote None Display the bot's voting link.
userinfo [mention] None Find a user's information.
serverinfo None Get information about the server.
botinfo None Get the bot's statistics.

Search

The commands used to search the internet.

bing [query*] None Search Bing.
yahoo [query*] None Search Yahoo.
ask [query*] None Search Ask.com.
